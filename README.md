# Mobile Price Classification Project

This project is part of a group machine learning project based on the **Mobile Price Classification** dataset. The goal is to predict the **price range** of a mobile phone using its technical features and hardware specifications.

## Objective
We aim to compare multiple machine learning models on the same dataset using the same preprocessing steps and evaluation setup. This allows us to fairly analyze which model performs best for mobile price classification.

## Models Used
The models for this project are divided among the group members as follows:
- **K-Nearest Neighbors (KNN)** 
- **Decision Tree**
- **Neural Networks**
- **Random Forest**

## Dataset
- **Dataset:** Mobile Price Classification
- **Task Type:** Classification
- **Target Variable:** Price range

## Experimental Setup
- **Train/Test Split:** 75% training, 25% testing
- **Random State:** 42

All models are trained and tested using the same split to ensure a fair comparison.

## Project Goal
The main goal of this project is to evaluate how well different classification models can predict the price category of a mobile phone and determine which model is the most effective for this dataset.
